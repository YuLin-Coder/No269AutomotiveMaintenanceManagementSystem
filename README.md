## 项目介绍
基于java+jsp的汽车维修保养管理系统：前端 jsp、jquery，后端 servlet、jdbc；角色分为管理员、用户；集成车主信息管理、保养管理、维修管理、年检保险管理、预约信息管理等功能于一体的系统。
